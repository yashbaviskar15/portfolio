## 🤝 Support & Help

### **If You Need To:**
- **Change colors**: Edit CSS variables in `:root` at top of `src/index.css`
- **Modify layout**: Edit component files in `src/components/`
- **Update content**: Edit text directly in component JSX
- **Add sections**: Create new component file and import in `App.jsx`
- **Change fonts**: Update `@import` in `src/index.css`

### **Common Issues:**
**Q: Links not working?**  
A: Update `href` values in component files

**Q: Images not showing?**  
A: Place images in `public/` folder and use `/filename` paths

**Q: Animations slow on mobile?**  
A: Reduce `animation-duration` for lower-end devices
